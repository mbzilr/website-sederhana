HTML Linking & Resources Made Simple

Learn how to link between pages and reference external files

Linking to Other Pages

To link to other HTML pages in the same directory, simply use the filename: 

<a href="about.html">About Us</a>
<a href="contact.html">Contact</a>

Using External Resources

To use CSS and JS files from subdirectories, reference them in your HTML: 

<!-- In your <head> section -->
<link rel="stylesheet" href="styles/main.css">
<script src="scripts/app.js"></script>

Note: The same method works for images:

<img src="images/logo.png" alt="Logo">

Folder Structure Example

Organize your project like this:

my-website/
├── index.html
├── about.html
├── contact.html
├── styles/
│   └── main.css
├── scripts/
│   └── app.js
└── images/
    ├── logo.png
    └── banner.jpg

Remember: Always use relative paths. If your HTML file is in a subfolder, you may need to use ../ to go up one directory level.
